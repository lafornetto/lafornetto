# La Fornetto

Official website for La Fornetto Restaurant, Pub & Pizzeria in Älvkarleby, Sweden.

Built with React, TypeScript and Vite.

## Features

- Responsive website
- Swedish and English language support
- Restaurant menu with categories, prices and allergens
- Lunch information
- News and campaign sections
- Catering information
- Gift card section
- Opening hours
- Contact information
- Google Maps integration
- Facebook, Instagram and WhatsApp links
- Editable homepage content through a separate admin panel
- Editable homepage images
- API connection for dynamic restaurant data
